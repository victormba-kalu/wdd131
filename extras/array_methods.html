map()

Returns a new array based upon a function. Each item of the source array is processed.

General form for the map() method

let newArray = originalArray.map(callbackFn, [thisArgument])

Example

Given the following array of numbers: const numbers = [1,2,3,4];

‚ùî Create a new array that contains each of the numbers array values at double the value.

Solution

let doubleNumbers = numbers.map(number => number * 2);

Result

doubleNumbers [2,4,6,8]

    filter()

    Returns a new array containing only the elements that past the given filter test. Each value of the original array is tested.

    General form for the .filter() method

    let newArray = originalArray.filter(callbackFn, [thisArgument])

    Example

    Given the following array of numbers: const numbers = [1,2,3,4,5,6,7];

    ‚ùî Create a new array that contains all the even numbers from the numbers array.

    Solution

    let evens = numbers.filter(number => number % 2 === 0);

    Result

    evens [2,4,6]

      reduce()

      Reduces an array by using auser-supplied "reducer". Each value of the original array is used and the reduction results a single value.

      General form for the .reducer() method

      let result = originalArray.reduce(callbackFn, [thisArgument])

      Example

      Given the following array of numbers:

      const numbers = [1,2,3,4,5,6,7];

      ‚ùî Given a base/initial value of 10, find the sum of all the values in the numbers array plus the base value.

      const initialValue = 10;

      Solution

      const sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, initialValue);

      Result

      sum 49

      forEach()

      Executes a provided function once for each array element. Unlike map(), it does not return a new array.

      General form for the forEach() method

      originalArray.forEach(callbackFn, [thisArgument])

      Example

      Given the following array of colors: const colors = ["red", "blue", "green", "yellow"];

      ‚ùî Log each color from the colors array to the console.

      Solution

      colors.forEach(color => console.log(color));

      Result

      Each color is printed in the console.

        find()

        Returns the first element in an array that satisfies the provided testing function. If no values match, it returns undefined.

        General form for the find() method

        let foundItem = originalArray.find(callbackFn, [thisArgument])

        Example

        Given the following array of numbers: const numbers = [10, 25, 40, 55, 70];

        ‚ùî Find the first number in the numbers array that is greater than 30.

        Solution

        let firstLargeNumber = numbers.find(num => num > 30);

        Result

        firstLargeNumber 40

        indexOf()

        Returns the index (location) of the first occurrence of a specified value in an array. If the value is not found, it returns -1. This makes it great for determining if something exists in the array or not.

        General form for the indexOf() method

        let index = originalArray.indexOf(searchElement, [fromIndex])

        Example

        Given the following array of names: const names = ["Alice", "Bob", "Charlie", "David"];

        ‚ùî Find the index of "Charlie" in the names array.

        Solution

        let charlieIndex = names.indexOf("Charlie");

        Result

        charlieIndex 2
